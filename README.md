<h1 align="center"> Landing Page </h1>
<br>

Este repositório apresenta uma landing page desenvolvida com HTML, CSS e JavaScript puro, construída como parte das aulas práticas da formação Desenvolvedor(a) Full Stack Web +praTI e Codifica.

A proposta é demonstrar na prática a criação de uma página moderna, responsiva e com foco na apresentação clara de um projeto fictício.

🔗 **Acesse a landing page no GitHub Pages**:  👉 [Clique aqui para visualizar](https://codingloria.github.io/landingPageMaisPraTI/)

## 🎯 Objetivos da Landing Page:
- Divulgar o projeto fictício.
- Apresentar as funcionalidades previstas.
- Treinar práticas de semântica, estilização e interação via JavaScript.

## 🌐 Tecnologias Utilizadas

- HTML
- CSS
- JavaScript 

## 🧩 Funcionalidades da Página

- Página responsiva com layout adaptado para diferentes telas
- Barra de navegação simples
- Botões interativos
- Rodapé com créditos
